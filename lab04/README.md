### Базовый датчик случайных чисел

**Задание:**
- реализовать базовый датчик случайных чисел;
- вычислить выборочные среднее и дисперсию:
  - для реализованного датчика;
  - для встроенного генератора языка программирования;
- размер выборки — **100 000 значений**;
- сравнить результаты с теоретическими;
- сделать вывод.

**Введение:**
Реализован мультипликативный конгруэнтный генератор, являющийся частным случаем линейного конгруэнтного генератора с параметром с=0

**Параметры**

```cpp
seed = generateSeed();
M = std::pow(2, 31) - 1;
b = 16807;
```
Параметры взяты из минимального стандарта МКГ (MINSTD), в качестве зерна используется текущее unix время в секунда с эпохи.

```cpp
int generateSeed()
{
    return (std::chrono::duration_cast<std::chrono::seconds>(std::chrono::system_clock::now().time_since_epoch()).count());
}
```

**Теория:**
- БД генерирует случаные числа в диапазоне [0, 1]. Среднее значение и дисперсия для него в теоории должны быть равны 0.5 и ~0.083 соотвественно. Значения моего датчика и встроенного:
```
Для MCG: Среднее: 0.4987899935, Дисперсия: 0.0830210435
Для MT: Среднее: 0.5000508277, Дисперсия: 0.0832516930
```
- Апериодичность: БД не может быть апериодичным по природе, но в МКГ период достаточно велик, чтобы отсутствие апериодичности не ощущалось. Период = M.
- Вопроизводимости: При задании одинокового зерна значения датчика будут повторяться
